Glossário de Tecnologia da Informação

Principais conceitos utilizados na área de TI.

HTML

Linguagem responsável pela estrutura de páginas web.

CSS

Linguagem utilizada para estilizar páginas web.

JavaScript

Linguagem de programação usada para criar interatividade.

API

Permite comunicação entre sistemas diferentes.

Banco de Dados

Local onde as informações são armazenadas.

Servidor

Computador que fornece serviços pela rede.

Git

Sistema de controle de versão.

GitHub

Plataforma para hospedar projetos Git.

Front-end

Parte visual com a qual o usuário interage.

Back-end

Parte responsável pelas regras de negócio e dados.
